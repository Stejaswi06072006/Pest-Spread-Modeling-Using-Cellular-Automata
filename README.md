# Pest Spread Modeling Using Cellular Automata

## Overview
This project simulates pest infestation across farmland using Cellular Automata.

## Features
- Grid-based farmland simulation
- Initial infection logic
- Wind effect
- Humidity effect
- Crop density effect
- Moore and Von Neumann neighborhoods
- Heatmap visualization
- Risk assessment

## Technologies Used
- Python
- Streamlit
- NumPy
- Matplotlib
